# docs/05_CODEX_INITIAL_PROMPT.md — Prompt initial à donner à Codex

Tu travailles dans un repo vierge déjà cloné localement. (celui ci)

Je veux créer un portfolio web ultra moderne avec React, TypeScript et GitHub Pages.

Avant de coder, lis obligatoirement :

* `AGENTS.md`
* `docs/00_PROJECT_BRIEF.md`
* `docs/01_TECHNICAL_ARCHITECTURE.md`
* `docs/02_UI_UX_DIRECTION.md`
* `docs/03_CONTENT_MODEL.md`
* `docs/04_BACKLOG.md`

Ensuite, réalise la Phase 1 et la Phase 2 du backlog.

Objectif de cette première itération :

1. Initialiser proprement le projet si nécessaire.
2. Mettre en place l’architecture React/TypeScript.
3. Créer les dossiers principaux.
4. Créer les fichiers de données avec placeholders.
5. Créer le layout global.
6. Créer le Header.
7. Créer le Footer.
8. Créer le routing.
9. Créer des pages temporaires pour :

   * Accueil ;
   * BrawlCards: Origins ;
   * Les Terres de Verdelya ;
   * Projets d’étude ;
   * CV ;
   * Hobbies ;
   * 404.
10. Ajouter les styles globaux et les design tokens.
11. Vérifier que `npm run build` fonctionne.

Contraintes :

* Ne pas chercher à tout finir dans cette première itération.
* Ne pas inventer les vrais liens sociaux.
* Utiliser des `TODO` propres pour les informations manquantes.
* Préparer une base solide et maintenable.
* Garder un style visuel déjà moderne dès le départ.
* Le site doit être compatible GitHub Pages.

À la fin, résume :

* les fichiers créés ;
* les choix techniques ;
* les commandes à lancer ;
* les prochaines étapes recommandées.
