# docs/04_BACKLOG.md — Backlog de développement

## Phase 1 — Initialisation

* Vérifier que le projet est bien initialisé.
* Installer React, TypeScript et Vite si nécessaire.
* Installer React Router.
* Créer la structure de dossiers.
* Créer les fichiers de données.
* Créer les styles globaux.
* Créer les design tokens.
* Configurer le build.

Critères d’acceptation :

* `npm run dev` fonctionne.
* `npm run build` fonctionne.
* La structure du projet est propre.
